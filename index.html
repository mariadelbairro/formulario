Questão 2

  1. Na tag input com atributo type="text", para que serve o atributo "placeholder"?

    2. R: O atributo placeholder é uma pequena frase ou palavra, destinada a ajudar os usuários a entender como preencher um determinado espaço reservado.

  2. Na tag input com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

    3. R: O atributo checked é usado para perguntas de Sim ou Não ( questoes de marcar x ), sendo representado por um botão marcado/desmarcado quando clicado.

  3. Na tag input para que serve o atributo "value"?

    4. R: O atributo value é utilizado para definir o valor a ser enviado por este item.

  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    5. R: input type= "radio" define um botão de rádio, os botões de opção permitem que um usuário selecione apenas uma de um número limitado de opções.

  5. Para que serve a tag label? Para que serve o atributo "for" nessa tag?

    6. R: Ela especifica qual o "rotulo" do input (a que se refere o input), e ajuda na experiência do usuário durante a utilização e preenchimento do formulário. O atributo for faz um tipo de link entre a label e o input. Logo o for tem que apontar para o ID do input.

  6. Na tag option para que serve o atributo "selected"?

    7. R: A tag dentro do elemento define as opcões disponiveis na lista suspensa.

  7. Nas tags de formulário para que serve o atributo "required"?

    8. R: É um atributo booleano.Quando presente, especifica que um campo de entrada deve ser preenchido antes de enviar o formulario.

  8. Na tag form para que serve o atributo "action"?

    9. R: O atributo action define a açao a ser executada quando o formulario for enviado.

  9. Na tag form para que serve o atributo "method"?

    10. R: O atributo method define qual o método HTTP para enviar os dados

  10. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?

    11. R: Atributo POST, isso porque esse atributo oculta os dados fornecidos no fomulario e os guarda internamente.

  11. Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?

    12. R:POST, visto que o metódo GET tem um limite de caracteres.
